1. Run CreateFolder.py to create folders for further use
2. Download .npy data file from this link(https://console.cloud.google.com/storage/browser/quickdraw_dataset/full/numpy_bitmap).
3. Run ViewDataset.py to view sample doodles from different classes
4. Run TrainAll.py to train all the models
5. Run Main.py to scan the doodle and predict based on different models
6. Run Saliency.py to obtain saliency map
